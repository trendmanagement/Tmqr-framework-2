set_net_position(date: datetime.datetime, net_position_dict: typing.Dict[tmqrfeed.contracts.ContractBase, typing.Tuple[[float, float], float]]) → None[source]¶

Set net position at given date (overwrites old position if it exists). This method intended to be used by low-level Quote* algorithms to initiate positions, generic strategies should use add_net_position() method.

This method allow to change position at the previous date, use with care this could ruin data validity.

Parameters:
  • date – datetime
  • net_position_dict – Dict[ContractBase, Tuple(decision_px, exec_px, qty)

class tmqrfeed.position.PositionReadOnlyView(datamanager, position_dict=None, **kwargs)[source]¶

Position read only view used to view precalculated Index position with decision_time_shift this class used as proxy to get position using original decision time of the instrument.

Only get_net_position() method allowed, other methods will raise PositionReadOnlyError()
